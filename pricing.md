# Pricing

*Transparent, pay-per-use pricing for distributed computing.*

<br>

## How Pricing Works

AetherGrid uses a token-based economy where computational resources are exchanged for Aether tokens. Unlike traditional cloud providers, you can both earn and spend tokens within the same ecosystem.

### Token Economics

- **Earn Tokens**: Contribute your idle computing resources to the network
- **Spend Tokens**: Use tokens to run your computational tasks
- **Trade Tokens**: Exchange tokens on supported cryptocurrency exchanges
- **Self-Sustaining**: Offset your computing costs by contributing resources during less busy hours

<br>

## Computing Costs

Pricing is based on actual resource consumption.

* When you submit a program to the grid, it's difficulty is computed on a benchmark system.
* Based on the computation difficulty and the length of time utilized, a price is calculated per system.

{: .note}
You only pay a single upfront amount. It is up to the computer clients to decide how much they will compute for the amount.

<br>

## Getting Started

### Option 1: Earn First

1. Install the AetherGrid client
2. Configure resource contribution
3. Start earning tokens automatically
4. Use earned tokens for your own computations

### Option 2: Purchase Tokens

1. Buy AetherGrid tokens on supported exchanges
2. Transfer tokens to your AetherGrid wallet
3. Submit computational tasks immediately
4. Pay only for what you use
